Pod Bay Doors is a simple little Javascript utility for animating div elements in a pod door like fashion. There is nothing too special going on, just some simple animation using jQuery and some positioning on a container div as well as the inside div elements.

For some extra sauce, I have included the jQuery Easing Plugin so that it is not limited to jQuery's "swing" ease. This is not required, but is a personal preference.

The only real requirement is a recent implementation of jQuery.

Tested in IE 8+, Chrome, and Firefox.
